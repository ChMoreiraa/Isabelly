# ❤️ Site Especial para Isabelly

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como uma homenagem especial para minha namorada, Isabelly.

Além do significado pessoal, o projeto serviu como uma oportunidade para praticar conceitos de desenvolvimento web, trabalhando com HTML, CSS e JavaScript para criar uma experiência interativa e personalizada.

A proposta foi transformar uma ideia simples em um projeto real, publicado e acessível pela internet.

---

## 🚀 Funcionalidades

* Página web personalizada.
* Interface interativa.
* Sistema de acesso por senha.
* Design customizado.
* Hospedagem através do GitHub Pages.

---

## 🛠️ Tecnologias Utilizadas

### Frontend

* HTML5
* CSS3
* JavaScript

### Ferramentas

* Git
* GitHub

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto foi unir aprendizado técnico com criatividade, desenvolvendo uma aplicação web funcional enquanto praticava conceitos fundamentais do desenvolvimento Frontend.

## 📚 Conceitos Aplicados

* Estruturação de páginas com HTML.

* Manipulação do DOM com JavaScript.

* Publicação de aplicações utilizando GitHub Pages.
